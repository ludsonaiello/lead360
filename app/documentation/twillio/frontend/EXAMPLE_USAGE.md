## Pattern

1. Open `PROMPT.md`
2. Find the line: `sprint-[N]-[name].md`
3. Replace with actual sprint file: `sprint-2-sms-configuration.md`
4. Copy entire prompt
5. Paste to AI developer in chat
6. Done! 🚀

**The prompt is only ~40 lines - super easy to use!**
